# 口袋城市首頁設計構想

## 方案一：都市夜景極簡主義
<response>
<idea>
**Design Movement**: 現代都市極簡主義（Urban Minimalism）

**Core Principles**:
- 深夜都市的靜謐感，以大量留白對比精緻細節
- 墨綠為主色調，金色作為點綴與強調
- 文字排版主導視覺層次，減少裝飾性元素
- 幾何線條與精準網格

**Color Philosophy**:
- 主背景：深墨綠 #1a2e1a / #0d1f0d
- 強調金色：#c9a227 / #e8c547
- 文字：米白 #f5f0e8
- 情感意圖：高端、穩重、值得信賴

**Layout Paradigm**:
- 非對稱分割版面，左側文字右側視覺
- Hero 區塊佔滿螢幕高度
- 產品卡片以錯位排列呈現

**Signature Elements**:
- 細金線分隔區塊
- 城市建築輪廓剪影作為裝飾
- 數字計數器展示平台規模

**Interaction Philosophy**:
- 滑鼠移動視差效果
- 卡片 hover 時金色邊框浮現
- 平滑滾動配合淡入動畫

**Animation**:
- 文字逐字打字效果
- 區塊進入視窗時從下方滑入
- 金色粒子背景輕微飄動

**Typography System**:
- 標題：Noto Serif TC（繁體中文）+ Playfair Display（英文）
- 內文：Noto Sans TC
- 強調：細體大字母英文
</idea>
<text>都市夜景極簡主義，深墨綠背景配金色細節，非對稱分割版面</text>
<probability>0.08</probability>
</response>

## 方案二：精工質感商務風
<response>
<idea>
**Design Movement**: 精工質感商務（Crafted Corporate）

**Core Principles**:
- 淺色背景搭配墨綠與金色的精緻點綴
- 強調「工具感」與「效率感」
- 卡片式模組化設計，清晰的資訊層次
- 品牌識別度高，適合 SaaS 平台定位

**Color Philosophy**:
- 主背景：暖白 #fafaf8
- 墨綠主色：#2d5a27 / #1e3d1a
- 金色輔助：#b8860b / #d4a017
- 情感意圖：專業、可靠、現代

**Layout Paradigm**:
- 全寬 Hero 橫幅，中央文字佈局
- 三欄產品展示區
- 交替左右圖文區塊

**Signature Elements**:
- 墨綠漸層 Hero 背景
- 金色 CTA 按鈕
- 圓角卡片配細陰影

**Interaction Philosophy**:
- 按鈕 hover 時金色光暈
- 卡片微微上浮效果
- 導航欄滾動後變色

**Animation**:
- 數字統計跳動效果
- 功能圖示旋轉進入
- 背景漸層緩慢移動

**Typography System**:
- 標題：Noto Serif TC + Cormorant Garamond
- 內文：Noto Sans TC
- 強調：大寫英文字母間距加寬
</idea>
<text>精工質感商務風，淺色背景配墨綠金色點綴，卡片式模組化設計</text>
<probability>0.07</probability>
</response>

## 方案三：沉浸式深色奢華風（選擇此方案）
<response>
<idea>
**Design Movement**: 沉浸式深色奢華（Immersive Dark Luxury）

**Core Principles**:
- 深色沉浸式體驗，讓內容成為焦點
- 墨綠底色配金色光澤，呈現高端 SaaS 質感
- 非線性視覺流動，引導用戶探索
- 精緻的微動效強化互動感

**Color Philosophy**:
- 主背景：極深墨綠 #0a1a0a / #0f2010
- 卡片背景：#152515 / #1a2e1a
- 金色主調：#c8a84b / #e2c068
- 金色光澤：#f0d080
- 文字：#e8e0d0（暖白）
- 情感意圖：奢華、科技、值得信賴、現代感

**Layout Paradigm**:
- 全螢幕沉浸式 Hero，背景有城市光點效果
- 產品以大型卡片橫向展示
- 特色區塊以斜切分割線區分
- 底部 CTA 區塊有金色漸層背景

**Signature Elements**:
- 金色細線邊框與分隔線
- 半透明玻璃態卡片（glassmorphism）
- 城市光點/星點背景動畫

**Interaction Philosophy**:
- 卡片 hover 時金色邊框發光
- 按鈕有金色光暈脈衝動畫
- 滾動觸發的淡入動畫

**Animation**:
- Hero 背景光點緩慢飄動
- 文字從下方淡入滑動
- 卡片進入視窗時縮放淡入
- 金色 CTA 按鈕持續脈衝

**Typography System**:
- 標題：Noto Serif TC（繁體中文）+ Cormorant Garamond（英文）
- 內文：Noto Sans TC
- 英文副標：字母間距加寬的細體
</idea>
<text>沉浸式深色奢華風，極深墨綠底色配金色光澤，玻璃態卡片與光點背景動畫</text>
<probability>0.09</probability>
</response>
